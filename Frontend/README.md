# 🚀 Interview AI

An AI-powered interview preparation platform that helps candidates prepare smarter by analyzing resumes, matching them with job descriptions, identifying skill gaps, and generating personalized interview reports.

## ✨ Features

* 📄 Resume Upload & Analysis
* 🎯 Job Description Matching
* 🤖 AI-Powered Interview Report Generation
* 📊 Skill Gap Analysis
* 📝 Personalized Interview Questions
* 🛣️ Learning Roadmap & Preparation Suggestions
* 🔐 Secure Authentication System
* 📱 Responsive Modern UI

---

### Frontend

### Authentication

* JWT Authentication

* Vercel (Frontend)

---

## 🎯 How It Works

1. Create an account or log in.
2. Upload your resume.
3. Paste the target Job Description.
4. Generate an AI-powered report.
5. Review:

   * Resume strengths
   * Skill gaps
   * Interview questions
   * Preparation roadmap
6. Start preparing with confidence.

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/Namitjoshii/interview-ai.git
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Environment Variables

Create a `.env` file:

```env
PORT=3000

MONGODB_URI=your_mongodb_connection_string

JWT_SECRET=your_jwt_secret

GEMINI_API_KEY=your_gemini_api_key
```
